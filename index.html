Twiga-Soko

Description

In this project the my main aim was to try and put into practice my basic understanding of react, how to import components, libraries and styles to a react project. This project is a clone of Twiga foods soko yetu landing page which is a carousel with different background images and containers in each slide with brief statements of the benefits of Soko yetu, my first approach was to make as slider my self using react use state but it didn't turnout as good as i wanted and so i had to use other option which was to use react-slick.

Link to the original website

Languages Used

  • React 66.9%
  • CSS 32%
  • HTML 1.1%

Problems Encountered

Being my first react basic project they weren't many challenges, the main one was to try and use the same technique that was used in the landing page of Soko Yetu where to make the images background images responsive, they had to set of each image, the desktop one and the mobile one. My first approach to make the background images appear for their respective screen resolution was to import each images and store them in a component which i would later import to my App.jsx and use react hooks useState and useEffect along side windows.innerWidth to determine in the width is greater or less than 768px if so the code triggers each component to render base on the conditions, this approach was good but the CSS was crashing, i resorted to using the css background image property to each slider and defining there responsiveness with @media.
